# Инструкция по добавлению ресурсов в Xcode проект

## Важно!
Файлы изображений и музыки были скопированы в папку `BirdEggs/Resources/`, но их нужно добавить в Xcode проект вручную.

## Шаги для добавления ресурсов:

### 1. Добавление изображений в Assets.xcassets (рекомендуемый способ):

1. Откройте проект в Xcode
2. Откройте `Assets.xcassets` в навигаторе проекта
3. Для каждого изображения:
   - Кликните правой кнопкой в Assets.xcassets → New Image Set
   - Переименуйте Image Set в имя файла (например, `bird` для `bird.webp`)
   - Перетащите файл `.webp` в соответствующий Image Set
   - Повторите для всех изображений

### 2. Альтернативный способ - добавление папки Resources:

1. В Xcode навигаторе кликните правой кнопкой на папку `BirdEggs`
2. Выберите "Add Files to BirdEggs..."
3. Выберите папку `Resources`
4. Убедитесь, что выбрано:
   - ✅ "Copy items if needed"
   - ✅ "Create groups" (не "Create folder references")
   - ✅ Target "BirdEggs" отмечен
5. Нажмите "Add"

### 3. Добавление музыки:

1. Перетащите файл `musick.mp3` из `Resources/Sounds/` в навигатор проекта Xcode
2. Убедитесь, что выбран Target "BirdEggs"
3. Файл должен появиться в проекте

## Проверка:

После добавления ресурсов, код автоматически будет использовать их вместо системных иконок SF Symbols.

## Структура ресурсов:

- **Изображения**: `BirdEggs/Resources/Images/` (все .webp и .png файлы)
- **Музыка**: `BirdEggs/Resources/Sounds/musick.mp3`

## Примечание:

Если ресурсы не будут добавлены в Xcode проект, приложение будет использовать fallback варианты (системные иконки SF Symbols и цветные фоны).
