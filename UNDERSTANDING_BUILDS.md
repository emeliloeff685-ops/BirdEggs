# Понимание билдов в App Store Connect

## Почему виден только последний билд в Versions?

Это **нормальное поведение** App Store Connect:

- **Каждая версия приложения может иметь только ОДИН активный билд**
- Когда вы загружаете новый билд для той же версии, он **заменяет** предыдущий
- Старый билд не удаляется, но становится неактивным

---

## Где можно увидеть все билды?

### 1. TestFlight → iOS Builds ⭐ (РЕКОМЕНДУЕТСЯ)

**Путь:**
```
App Store Connect → My Apps → Bird Eggs → TestFlight → iOS Builds
```

**Что вы увидите:**
- **ВСЕ загруженные билды** (даже неактивные)
- Версию каждого билда (например, 1.0.18)
- Статус обработки
- Дата загрузки
- Размер билда

**Это лучшее место для просмотра всех билдов!**

---

### 2. Activity (Журнал действий)

**Путь:**
```
App Store Connect → My Apps → Bird Eggs → Activity
```

**Что вы увидите:**
- История всех загрузок
- Все действия с билдами
- Ошибки (если были)
- Время загрузки каждого билда

---

### 3. App Store → Versions

**Путь:**
```
App Store Connect → My Apps → Bird Eggs → App Store → Versions
```

**Что вы увидите:**
- Только **активный билд** для каждой версии
- Если версия не создана, билды будут в TestFlight

---

## Как работают версии и билды?

### Версия приложения (Version)
- Это номер версии, который видят пользователи (например, 1.0)
- Устанавливается в Xcode: `MARKETING_VERSION = 1.0`

### Билд (Build)
- Это номер сборки (например, 1, 2, 3...)
- Устанавливается в Xcode: `CURRENT_PROJECT_VERSION = 1`
- Каждая версия может иметь несколько билдов, но активен только один

### Пример:
```
Версия 1.0:
  - Билд 1 (старый, неактивный)
  - Билд 2 (старый, неактивный)
  - Билд 3 (текущий, активный) ← виден в Versions
```

Все три билда будут видны в **TestFlight → iOS Builds**!

---

## Как создать новую версию для нового билда?

Если вы хотите, чтобы новый билд был в отдельной версии:

1. **В Xcode:**
   - Увеличьте `MARKETING_VERSION` (например, с 1.0 до 1.1)
   - Или увеличьте `CURRENT_PROJECT_VERSION` (например, с 1 до 2)

2. **В App Store Connect:**
   - Создайте новую версию вручную
   - Или она создастся автоматически при загрузке билда с новым номером версии

## Частые вопросы:

### Q: Почему я вижу только один билд в Versions?
**A:** Это нормально. Каждая версия может иметь только один активный билд. Все билды видны в TestFlight.

### Q: Где найти все мои билды?
**A:** TestFlight → iOS Builds - там видны все загруженные билды.

### Q: Как сохранить старый билд?
**A:** Старые билды не удаляются, они просто становятся неактивными. Они все еще видны в TestFlight.

### Q: Можно ли использовать старый билд?
**A:** Да, в TestFlight можно выбрать любой билд для тестирования, даже неактивный.
